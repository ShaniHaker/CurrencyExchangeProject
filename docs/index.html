Currency Exchange SDK

GitHub JitPack Live API Backend Repo

An Android SDK that provides simple Kotlin methods for live currency exchange rates, conversion, favorites, and history — powered by a Flask + MongoDB backend deployed on Railway.

Installation

1. Add JitPack to settings.gradle.kts

dependencyResolutionManagement {
    repositories {
        google()
        mavenCentral()
        maven { url = uri("https://jitpack.io") }
    }
}

2. Add the dependency

dependencies {
    implementation("com.github.ShaniHaker:currencysdk:1.0.1")
}

Usage

import com.example.currencysdk.CurrencySdk

val sdk = CurrencySdk()

viewModelScope.launch {
    // Get supported currencies
    val currencies = sdk.getSupportedCurrencies()

    // Get live exchange rate
    val rate = sdk.getExchangeRate("USD", "ILS")

    // Convert an amount
    val result = sdk.convertAmount("USD", "ILS", 100.0)

    // Favorites
    sdk.addFavorite("USD", "ILS")
    val favorites = sdk.getFavorites()
    sdk.deleteFavorite(favorites.favorites[0].id)

    // History
    val history = sdk.getHistory()
}

SDK Methods

MethodDescriptionReturns
getSupportedCurrencies()List all supported currenciesCurrenciesResponse
getExchangeRate(from, to)Live exchange rateRateResponse
convertAmount(from, to, amount)Convert and save to historyConversionResponse
getFavorites()All saved favoritesFavoritesResponse
addFavorite(from, to)Save a currency pairMessageResponse
deleteFavorite(id)Remove a favoriteMessageResponse
getHistory()All past conversionsHistoryResponse

API Endpoints

MethodEndpointDescription
GET/currenciesSupported currencies
GET/rates?from=USD&to=ILSLive exchange rate
GET/convert?from=USD&to=ILS&amount=100Convert amount
GET/favoritesList favorites
POST/favoritesSave favorite
DELETE/favorites/<id>Delete favorite
GET/historyConversion history
